Lesson 2 :: Layout

簡單來說就是幫網頁穿衣服，幫一個網站完成視覺上的美感。

我會將編成網頁風格的方式分為兩類：

  1. Hard Code - 直接寫入網頁內部的方式
  2. Import CSS file - 導入CSS文件至每一頁的方式

本篇主要講解如何寫出自己網站的Style。

Mentor Jesse: 香港人心急嘛...

Hard Code Style :: 直入式

  1. <head>
  2.   <style type="text/css">
  3.   .red-box {
  4.     background-color: #f00;
  5.     color: #fff;
  6.   }
  7.   #green-box {
  8.     background-color: #0f0;
  9.     color: #fff;
  10.   }
  11.   </style>
  12. </head>

CSS - Cascading Style Sheets :: 導入式

HTML file

  1. <head>
  2.   <link rel="stylesheet" type="text/css" href="{{style}}.css" />
  3. </head>

請養成隨手關門的好習慣。

CSS file

  1. .red-box {
  2.   background-color: #f00;
  3.   color: #fff;
  4. }
  5. #green-box {
  6.   background-color: #0f0;
  7.   color: #fff;
  8. }

How to use? :: 應用

建立了一個簡單的CSS格式編碼後，我們應該要如何放進HTML裡頭使用呢？

閱讀以下範例：

  1. <div class="red-box">
  2.   This is the red box effect with class(.)
  3. </div>
  4. <div id="green-box">
  5.   This is the green box effect with ID(#)
  6. </div>

按照以上的編碼，你試試會得到一個怎麼樣的結果。

顏色的基本用法

色值編碼方式

  1. color: #ffffff - 十六進制式，R(紅)/00-0F ~ F0-FF/ G(綠)/00-0F ~ F0-FF/ B(藍)/00-0F ~ F0-FF/
  2. color: rgba(255,255,255,1) - 十進制式, R/0-255/ G/0-255/ B/0-255/ A(透明度)/0.0-1.0/
  3. color: white - 如果知道顏色的名字可以直接輸入 *不建議*
  4. 屬性 background 同上
